# 📦 Carpeta de Descargas

## ¿Qué va aquí?

Esta carpeta contiene el archivo APK de la aplicación para Android.

## 📱 Archivo necesario:

- **AetherCubix.apk** - El archivo APK generado por PWABuilder

## 🔧 Instrucciones:

1. **Copia el APK** que descargaste de PWABuilder
2. **Pégalo en esta carpeta** con el nombre `AetherCubix.apk`
3. **Sube los cambios** a tu hosting (Netlify/Vercel)

## ✅ Verificación:

Después de subir, el APK debe ser accesible en:
```
https://tu-dominio.com/downloads/AetherCubix.apk
```

## 📋 Notas:

- El archivo debe llamarse exactamente: `AetherCubix.apk`
- Tamaño aproximado: 1.6 MB
- Versión actual: 1.0.0
- Package name: com.aethercubix.app

## 🚀 Para actualizar:

Cuando tengas una nueva versión del APK:
1. Reemplaza el archivo `AetherCubix.apk`
2. Sube los cambios
3. Los usuarios descargarán automáticamente la nueva versión
