# Iconos PWA

Coloca aquí los iconos generados con el generador de iconos.

## Tamaños necesarios:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png (obligatorio)
- icon-384x384.png
- icon-512x512.png (obligatorio)

## Cómo generar:

1. Abre `/public/icon-generator.html` en tu navegador
2. Personaliza el texto y colores
3. Descarga todos los iconos
4. Guárdalos en esta carpeta

## Alternativa rápida:

Si no tienes un logo, puedes usar el generador incluido o crear iconos simples con:
- https://www.pwabuilder.com/imageGenerator
- https://favicon.io/favicon-generator/
