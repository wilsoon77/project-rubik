# Screenshots PWA

Coloca aquí capturas de pantalla de tu aplicación para mostrar en la tienda de apps.

## Tamaños recomendados:

### Móvil (portrait):
- home.png: 540×720px o 1080×1920px

### Desktop (landscape):
- productos.png: 1280×720px o 2560×1440px

## Cómo crear:

1. Abre tu app en el navegador
2. Usa las DevTools para emular dispositivos móviles
3. Toma capturas de las pantallas principales:
   - Página de inicio
   - Catálogo de productos
   - Carrito de compras
   - Checkout

Estas capturas se mostrarán cuando los usuarios instalen la PWA.
